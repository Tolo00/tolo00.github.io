48h Game Project

Flowermania

This game was made in 48 hours during a local game development competition. I worked together with two other developers who did all the graphical elements, while I focused on developing the gameplay loop.

Two primary features I worked on are the inventory system and the “Combinator”, used for combining two flowers into a new one.
To make these systems more modular I used Scriptable Objects as a data container to allow the separation between each flower state (seed, flower, inventory item) and the data associated with that specific flower (animations, sprites).
This way every flower state can use the data in any way it wants without introducing unnecessary coupling.

Tools Used

  • C#
  • Unity Game Engine
  • Scriptable Objects
  • Github (for version control)
