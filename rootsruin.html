Card Game

Roots & Ruin

This project is a card game developed for a duration of 2 months in conjunction with the course Gameproject 1. The game was made from scratch using the Unity Game Engine together with a team of 6 people.

I was responsible for the technical challenges associated with creating a card game. This included designing gameplay systems and tools in a sustainable way to ensure development could continue for a longer period if required.

As the game contains a lot of data associated with each card. I had to develop tools to make it easy for the designer to create and customize different card behaviour. To do this, I used Scriptable Objects and Custom Editors to create a custom interface you could use to customize the cards stats and behaviour.

Additionally, I played a key role in organizing the Unity project to ensure it was as user-friendly as possible. This involved establishing clear guidelines and teaching colleagues best practices to maintain a clean and efficient project structure.

Tools Used

  • C#
  • Unity Game Engine
  • State Machine
  • Scriptable Objects
  • Custom Editor
  • Github (for version control)
